////Aplicação
	O programa dá inicio pelo Login, no entanto como ainda não deve ter conta na nossa aplicação vamos começar pelo registo, se tiver então por favor salte este passo.
	Começos por clicar no botão de Registar, de seguida preencha todos os dados nos campos que aparecem no ecrã.
	Clique novamente no botão de Registar e se voltou para o menu de Login então parabéns e bem-vindo a nossa App. A partir deste momento encontra-se registado na aplicação.
	Sendo a ação realizada com sucesso irá para o menu principal da aplicação.
	Preencha os campos com dados do torneio e rapidamente ficará disponível.
	Dentro de Torneio existem 5 menus: Equipas, Jogadores, Jogos, Estatisticas, Informação de torneio.
	Poderá voltar para esta página pelo Icon no canto superior direito.

	Comecemos por criar uma equipa 
		Para criar uma equipa é necesario criar no botão onde tem um emblema e logo de seguida clicar no botão 'mais' onde o utilizador insere os dados da equipa.
		Logo após inserir a equipa, o utilizador pode ver a equipa inserida no menu das equipas. 
		No menu das equipas também podemos ver a informação das mesmas clicando em cima delas.

	Criar jogador
		Para criar um jogador é necessario criar no botão onde tem uma camisola e logo de seguida clicar no botão 'mais' onde o utilizador insere os dados do jogador e de seguida 
		clicar no botão 'mais' onde o utilizador insere os dados do jogador.
		Logo após inserir o jogador, o utilizador pode ver o jogador inserido no menu dos jogadores.
		No menu dos jogadores também podemos ver a informação dos mesmos clicando em cima deles.
 
	Criar jogo
		Para criar um jogo é necessario cliar no botão onde tem uma agenda com uma bola e clicar no botao 'mais' onde o utilizador insere os dados do jogo.
		Logo após inserir o jogo, o utilizador pode ver o jogo inserido no menu dos jogos.
		No menu dos jogos também podemos criar estatisticas clicando em cima do jogo, onde de seguida aparecerá uma formulario para inserir os dados da estatistica.	
	
	Ver estatistica
		Para ver as estatisticas é necessario clicar no botão das estatisticas que é logo primeiro e selecionar o jogo que quer ver as estatisticas.
	
	Ver informações do torneio
		Para ver as informações dos torneios basta clicar no bota com um 'I' no menu e o utilizador poderá ver a informação dos torneios assim como as médias se clicar no botão médias que tem nas 
		informações dos torneios.
	
	Ver dados pessoais	
		Para ver os dados pessoais basta clicar em cima no lado esquerdo da aplicação, onde tem a foto do utilizador. Quando clicar, o utilizador poderá ver as suas próprias informacões.
